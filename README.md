# Customer-churn-prediction-using-ML

The project is about Customer churn prediction , whether a person is staying or not

In this dataset
   * CreditScore        
   * Geography          
   * Gender              
   * Age                
   * Tenure                
   * Balance          
   * NumOfProducts etc ...
# Tools used :
  * pandas
  * numpy
  * matplotlib
  * seaborn
  * sklearn

Among all models, Random Forest and adaboost performed best because it handles nonlinear relationships and reduces overfitting using ensemble learning.
Logistic Regression gave stable baseline results.
SVM required proper scaling and hyperparameter tuning.
Naive Bayes performed moderately due to independence assumption.
KNN performance depended on optimal K value."""
